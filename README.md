# Multiplayer Quiz - Frontend 🎮

The client-side application for the multiplayer quiz game, built with **Vite**, **React**, **TypeScript**, and **Socket.io-client**.

## 🚀 Setup Instructions

1. **Install Dependencies:**

   npm install

2. **Configure Environment Variables:**

Change the name of the ".env.example" file to ".env" and add your variables following the examples.

(Note: You need a network url to access the website from anywhere that isn't the machine where the app is running, for example, a phone on the same WI-FI network).

3.  **Run Development Server:**

npm run dev

🔗 Connection

This frontend connects to the [Backend API](github.com/nicolasmagnabosco/multiplayer-quiz-back)

🛠 Tech Stack

    Framework: Vite + React

    Language: TypeScript

    Communication: Socket.io-client
